## Tech Stack
- Language: Python 3.10

- Database: PostgreSQL
- Containerization: Docker
- Deployment: Vercel, Docker
- Cloud Technology: Amazon Web Service (AWS)
- IaC: Cloud Formation in AWS
- CI/CD: GitHub Actions
